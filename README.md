# Send-JSON-Data-to-API
This is a Nodejs console app to send json data to specific endpoint(API). Read .JSON file content and send data to API one by one.
- Console log every data.

## Features
- Console log every JSON data.
- Fast speed.
- Minimun user involment.
- Console app

## Technology

- Nodejs

## Installation
Install dependencies:
```sh
npm install
```
Start:
```sh
node index
```

## Dependencies
- fs
- axios
